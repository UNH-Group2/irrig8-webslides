Irrig8 - Rachio Controller Interface App

Functionally the project stayed true to the original proposal. From a technical perspective, the goal was to create application that would interface with an IoT devivce and demonstrate a behavior change in the phycial world.

The Rachio controller provides a robust API and an interace that is completely wireless. The app function is to integrate with Rachio, discover any registered devices, control the zones and keep a history of individual zone actvitiy

Design Process

Technologies Used & How They Work

  • Passport - Multiple authentication options support
  • OAuth2 - Rachio Single Sign on authentication
  • Crypto - Node package for hashing passwords and keys
  • Redis - Web Cache for user identity
  • Express/Handlebars - Node based web server and UI framework
  • MySQL - Relational Database for Rachio config and activity
  • Sequelize - ORM Mapping to mySQL backend
  • Dotenv - Secured credentials allow for local & production deployments
  • Moment - Time utility library for zone update timestamps
  • Eslint - code rules for coding standards and practices
  • Heroku - Cloud deployment through GitHub and Travis CI
  • WebSlides - This HTML/CSS presentationnware
